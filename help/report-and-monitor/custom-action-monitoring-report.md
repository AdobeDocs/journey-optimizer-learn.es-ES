# Monitorización de acciones personalizadas

Explore cómo monitorizar de forma eficaz el estado operativo y el rendimiento de las llamadas HTTP realizadas desde sus recorridos a puntos finales de terceros mediante acciones personalizadas. Obtenga visibilidad de los errores de HTTP, los cuellos de botella de latencia y los problemas de límite que pueden afectar al envío.

Este informe ayuda a identificar problemas, mejorar el rendimiento del recorrido y garantizar un funcionamiento sin problemas.

Consulte la [documentación del producto](https://experienceleague.adobe.com/es/docs/journey-optimizer/using/configure-journeys/action-journeys/reporting) para obtener más información sobre esta función.
